GIT傻瓜试操作

发布于：2020-03-05|阅读：3500|标签：#GIT#HTML
Git初始化：
git config--global user.name'github用户名'
git config--global user.email'github邮箱'
初始化git：
git init
下载/克隆代码：
git clone仓库地址
提交到缓存区
git add文件名（*更新所有.上传所有）
添加说明到本地仓库
git commit-m'说明内容'
上传远程仓库打Tag
git tag tag名称
添加到远程仓库
git push
Tags提交
git push--tags
回到之前Tag提交状态
git log git reset--hard tag哈希值
删除文件
git rm文件名
查看当前状态
git status
查看远程
Tag git checkout分支名
创建新分支并切换
git checkout-b分支名
查看分支
git branch
切换分支
git checkout分支名
合并分支
git merge分支名
上传指定远程仓库
git remote add origin仓库地址
上传分支到远程仓库
git push-u origin分支名
